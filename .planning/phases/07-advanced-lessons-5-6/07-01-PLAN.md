# Plan 07-01: Advanced Lessons 5-6 - Prompting Techniques

**Phase**: 07-advanced-lessons-5-6
**Plan**: 07-01
**Objective**: Teach children effective AI communication through prompting techniques and practical exercises

## Context

Phase 7 shifts from understanding AI to actively using it. After Lessons 1-4 established foundational knowledge, Lessons 5-6 teach children how to communicate effectively with AI through prompting techniques. This prepares them for Lesson 7 where they'll build their own mini-apps.

**What exists**:
- Phase 3-6 foundation: Lesson framework, quiz system, progress tracking, core AI knowledge
- Children now understand: what AI is, types of AI, capabilities, and everyday applications

**What's needed**:
- Lesson 5: Prompting fundamentals (0-shot, few-shot, step-by-step)
- Lesson 6: Practical prompting exercises for different scenarios
- Quizzes that test prompting comprehension
- Child-appropriate examples of good vs. bad prompts

## Success Criteria

**From ROADMAP.md Phase 7**:
1. User can complete Lesson 5 "Mit KI sprechen" understanding prompt techniques (0-shot, few-shot)
2. User can complete Lesson 6 practice exercises for research and prompting
3. User understands difference between clear and vague prompts through interactive examples
4. User can apply prompting techniques in guided practice scenarios

## Implementation Plan

### Task 1: Lesson 5 - "Mit KI sprechen" (Prompting Fundamentals)

**File**: `src/data/lessons/lesson-5.json`

**Educational approach**:
- Move from passive understanding to active skill-building
- Introduce prompting as "talking effectively with AI"
- Use child-relatable analogies (asking teacher vs. asking friend)

**Content structure**:
- **Intro**: Importance of HOW you ask questions
- **Explanation 1**: Clear vs. vague questions
  - Bad: "Erzähl mir was" (too vague)
  - Good: "Erzähl mir eine kurze Geschichte über einen Hund" (specific)
  - Principle: Specificity improves results
- **Explanation 2**: Few-Shot prompting (giving examples)
  - Concept: "Show the AI what you want"
  - Example: "Schreib einen Witz wie: Warum ist die Banane krumm?"
  - Child-friendly term: "Beispiele geben"
- **Explanation 3**: Step-by-step decomposition
  - Complex tasks → small steps
  - Example: Planet question broken into 3 parts
  - Principle: "Kleine Schritte führen zum Ziel"
- **Summary**: Three key techniques recap

**Kiki integration**:
- Curious (intro) → Thoughtful (clear questions) → Happy (examples) → Proud (step-by-step) → Proud (summary)

**Pedagogical notes**:
- Avoid jargon like "zero-shot" in UI (explain concept, not term)
- Use "Few-Shot" because it sounds playful and memorable
- Reinforce AI Leadership: "Du sagst der KI genau, was du willst"

### Task 2: Lesson 6 - "Übungen" (Practice Exercises)

**File**: `src/data/lessons/lesson-6.json`

**Educational approach**:
- Apply Lesson 5 concepts to real scenarios
- Show progression: good → better prompts
- Build confidence through guided examples

**Content structure**:
- **Intro**: Practice time with Kiki
- **Exercise 1**: Story generation
  - Good: "Schreib eine kurze Geschichte über einen mutigen Hasen"
  - Better: "Schreib eine Geschichte über einen Hasen, der einen Schatz findet. Die Geschichte soll lustig sein."
  - Learning: More details = better output
- **Exercise 2**: Explanations
  - Good: "Was ist ein Regenbogen?"
  - Better: "Erklär mir, was ein Regenbogen ist. Erklär es so, dass ein Kind es versteht."
  - Learning: Specify audience/style
- **Exercise 3**: Idea generation
  - Good: "Gib mir Spielideen"
  - Better: "Gib mir 5 Spielideen für draußen, die man zu zweit spielen kann"
  - Learning: Numbers + constraints = useful results
- **Summary**: Three practice scenarios mastered

**Kiki integration**:
- Surprised (intro) → Happy (stories) → Thoughtful (explanations) → Curious (ideas) → Proud (summary)

**Pedagogical notes**:
- Each exercise shows progression (not just "right" answer)
- Encourages experimentation: "Probier es später mal aus!"
- Prepares for Lesson 7 mini-apps where they'll use these skills

### Task 3: Quiz 5 - "Mit KI sprechen"

**File**: `src/data/quizzes/quiz-5.json`

**Quiz structure** (5 questions):
1. **Clear vs. vague questions**: Identify better prompt
2. **Few-Shot definition**: What does it mean to show examples?
3. **Step-by-step strategy**: When to break down tasks
4. **Importance of clarity**: Why are clear questions better?
5. **Practical application**: Best way to ask for a joke

**Question design**:
- Multiple choice with child-friendly distractors
- Positive feedback: "Genau! Je klarer deine Frage, desto besser die Antwort."
- Corrective feedback: "Die genaue Frage ist besser: 'Erzähl mir eine Geschichte über einen Drachen'."

**Learning reinforcement**:
- Q1-Q2: Recognition (identify good prompts)
- Q3-Q4: Understanding (explain why techniques work)
- Q5: Application (construct own prompt)

### Task 4: Quiz 6 - "Übungen"

**File**: `src/data/quizzes/quiz-6.json`

**Quiz structure** (5 questions):

2. **Explanation prompting**: What to specify for age-appropriate explanations
3. **Idea generation**: Best structured question with numbers

**Question design**:

**Learning reinforcement**:

## Pedagogical Strategy

**Bloom's Taxonomy progression**:
- Lesson 5: **Understand** prompting concepts
- Lesson 6: **Apply** techniques to scenarios
- Quiz 5-6: **Analyze** good vs. bad prompts
- Lesson 7 (next): **Create** with AI using learned skills

**Constructivist approach**:
- Build on prior knowledge (Lessons 1-4)
- Active learning through examples
- Scaffolding: simple → complex prompts
- Metacognition: "Why does this work better?"

**Key learning outcomes**:
1. **Conceptual**: Understand that prompt quality affects output
2. **Procedural**: Apply clear/specific/example-based prompting
3. **Metacognitive**: Reflect on what makes prompts effective
4. **Affective**: Feel confident communicating with AI
